# 压缩与解压

## gzip / gunzip

> gzip：用于压缩文件。
>
> gunzip：用于解压文件。

### 基本语法

~~~
gzip file (压缩文件，只能将文件压缩为 .gz 格式)
gunzip file (解压文件)
~~~

## zip / unzip 

> zip：用于压缩文件。
>
> unzip：用于解压文件。

### 基本语法

~~~
zip [选项] xx.zip
unzip [选项] xx.zip
~~~

### 常用选项

~~~
zip:
	-r: 递归压缩，即压缩目录。
unzip:
	-d<目录>: 指定解压后文件的存放目录。
~~~

## tar

> tar：
> 	打包指令，打包的文件格式为 *.tar.gz。

### 基本语法

~~~
tar [选项] xx.tar.gz 打包的内容。
~~~

### 选项说明

| 选项 | 功能                |
| ---- | ------------------- |
| -c   | 产生 *.tar 打包文件 |
| -v   | 显示详细信息        |
| -f   | 指定压缩后的文件名  |
| -z   | 打包同时压缩        |
| -x   | 解压 .tar 文件      |
